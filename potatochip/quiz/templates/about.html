About PotatoChip

PotatoChip is an interactive quiz application built with Django, designed to make learning and testing knowledge fun and engaging. Whether you're a teacher creating quizzes for your students, a student joining a quiz session, or just someone who loves trivia, PotatoChip has you covered!

Features

  • Create Custom Quizzes: Design your own quizzes with multiple-choice questions. Set up questions, choices, and correct answers easily through our intuitive interface.
  • Join Live Quizzes: Enter a 6-digit quiz code to join an active quiz session. Answer questions in real-time and see your results instantly.
  • Real-Time Interaction: Powered by Django Channels and WebSockets, PotatoChip provides live updates during quiz sessions, making it perfect for classrooms or group activities.
  • User Authentication: Secure login and registration system to manage your quizzes and track your participation.
  • Responsive Design: Designed for seamlessly use on both desktop and mobile devices.

How to Use

  1. Sign Up: Create an account to start creating and joining quizzes.
  2. Create a Quiz: Click "Create a Quiz" to build your quiz. Add questions with multiple choices and mark the correct answers.
  3. Host Your Quiz: Once created, get a unique quiz code to share with participants.
  4. Join a Quiz: Enter the 6-digit code on the "Join a Quiz" page to participate.
  5. Enjoy: Answer questions as they appear and see your score at the end!

Technology Stack

PotatoChip is built using modern web technologies:

  • Backend: Django (Python web framework)
  • Database: SQLite (for development), easily configurable for PostgreSQL in production
  • Real-Time Features: Django Channels with Redis
  • Frontend: HTML, CSS, JavaScript with Bootstrap for styling
  • Deployment: Deployed on Digital Ocean

About the Name

PotatoChip is named after the adorable American Goldfinch which is known to have a call that sounds like "Po ta to chip". (Spoiler: their call sounds nothing like that)
